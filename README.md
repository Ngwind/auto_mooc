# auto_mooc

## 前置条件
1. python3
2. selenium库
3. chromedriver.exe
4. Chrome浏览器

安装教程可以参考：https://ngwind.github.io/2018/09/20/selenium-%E7%8E%AF%E5%A2%83%E5%87%86%E5%A4%87/

## 使用方法

在命令行用`python auto_mooc.py`运行。

有两个地方要自己输入：
 - 手动输入验证码，点击登录 
 - 手动点开第一个要刷的课程

其余详细方法自己摸索...
